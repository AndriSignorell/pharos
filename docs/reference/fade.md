# Fade Colors

Apply transparency and remove the alpha channel afterwards.

## Usage

``` r
fade(col, opacity = 0.5)
```

## Arguments

- col:

  vector of valid R colors

- opacity:

  opacity value between 0 and 1

## Value

Character vector of colors.
