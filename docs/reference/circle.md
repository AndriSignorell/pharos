# Circle Geometry

Create a circular geometry.

## Usage

``` r
circle(x = 0, y = 0, radius = 1, numPoints = 100)
```

## Arguments

- x, y:

  centre coordinates.

- radius:

  circle radius.

- numPoints:

  number of points used to approximate the circle.

## Value

An object inheriting from class `"circleGeometry"`.
