# Convert RGB to CMY

Convert RGB colors to the CMY color space.

## Usage

``` r
rgbToCmy(col, maxColorValue = 1)
```

## Arguments

- col:

  RGB matrix or hexadecimal colors.

- maxColorValue:

  Maximum channel value.

## Value

Numeric CMY matrix.
