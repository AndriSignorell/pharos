# Convert Hex Colors to RGB

Convert hexadecimal color strings to an RGB matrix.

## Usage

``` r
hexToRGB(hex)
```

## Arguments

- hex:

  Character vector of hexadecimal colors.

## Value

Integer matrix with RGB rows.
