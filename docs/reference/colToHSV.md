# Convert R Colors to HSV

Convert any valid R color specification to HSV.

## Usage

``` r
colToHSV(col, useAlphaChannel = FALSE)
```

## Arguments

- col:

  vector of valid R colors.

- useAlphaChannel:

  logical indicating whether the alpha channel should be included.

## Value

Numeric HSV matrix.
