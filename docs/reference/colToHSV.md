# Convert R Colors to HSV

Convert any valid R color specification to HSV.

## Usage

``` r
colToHSV(col, alpha = FALSE)
```

## Arguments

- col:

  Vector of valid R colors.

- alpha:

  Logical indicating whether alpha values should be returned.

## Value

Numeric HSV matrix.
