# Convert CMYK to RGB

Convert CMYK colors to RGB.

## Usage

``` r
cmykToRgb(col, maxColorValue = 1)
```

## Arguments

- col:

  numeric CMYK matrix (columns C, M, Y, K).

- maxColorValue:

  maximum channel value.

## Value

Numeric RGB matrix.
