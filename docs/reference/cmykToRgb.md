# Convert CMYK to RGB

Convert CMYK colors to RGB.

## Usage

``` r
cmykToRgb(cyan, magenta, yellow, black, maxColorValue = 1)
```

## Arguments

- cyan:

  Cyan values.

- magenta:

  Magenta values.

- yellow:

  Yellow values.

- black:

  Black values.

- maxColorValue:

  Maximum channel value.

## Value

Hexadecimal RGB colors.
