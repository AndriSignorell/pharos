# Regular Polygon Geometry

Create a regular polygon.

## Usage

``` r
regPolygon(x = 0, y = 0, radius = 1, numVertices = 6, startAngle = 0)
```

## Arguments

- x, y:

  Centre coordinates.

- radius:

  Circumradius.

- numVertices:

  Number of vertices.

- startAngle:

  Staring angle in radians.

## Value

An object inheriting from class `"regPolygonGeometry"`.
